# Generador de Enlaces de Streaming - RaulFlix\n\n## Descripción\n\nEl Generador de Enlaces automatiza la extracción de tokens y URLs de streaming desde servidores externos, eliminando la necesidad de procesos manuales.\n\n## Arquitectura\n\n```\nUsuario → Frontend → Backend API → Servidor Origen → Token → URL Streaming\n```\n\n## Componentes\n\n### Backend\n\n#### 1. LinkGenerator Service (`/backend/src/services/linkGenerator.js`)\n- **Función**: Extrae tokens de servidores externos\n- **Cache**: 15 minutos de duración\n- **Patrones**: Detecta múltiples formatos de video (m3u8, mp4)\n\n#### 2. Streaming Routes (`/backend/src/routes/streaming.js`)\n- `POST /api/streaming/generate-link` - Generar enlace individual\n- `POST /api/streaming/generate-multiple` - Generar múltiples enlaces\n- `GET /api/streaming/verify/:movieId` - Verificar validez del enlace\n- `GET /api/streaming/cache/stats` - Estadísticas del cache\n\n### Frontend\n\n#### 1. StreamLinkGenerator Component\n- **Ubicación**: `/frontend/src/components/StreamLinkGenerator.js`\n- **Función**: Interfaz para generar enlaces automáticamente\n- **Estados**: Loading, Success, Error\n\n#### 2. Player Integration\n- **Ubicación**: `/frontend/src/pages/Player.js`\n- **Función**: Integra el generador con el reproductor\n- **Flujo**: Seleccionar contenido → Generar enlace → Reproducir\n\n#### 3. Admin Panel\n- **Ubicación**: `/frontend/src/pages/admin/StreamingAdmin.js`\n- **Función**: Gestionar servidores y probar enlaces\n\n## Uso\n\n### 1. Reproducir Contenido\n\n1. Navegar a una película o serie\n2. Hacer clic en \"Generar Enlace\"\n3. El sistema extrae automáticamente el token\n4. Se reproduce el contenido\n\n### 2. Administración\n\n1. Acceder al panel de administración\n2. Agregar/quitar servidores de origen\n3. Probar enlaces individuales o múltiples\n4. Monitorear estadísticas del cache\n\n## API Endpoints\n\n### Generar Enlace Individual\n```javascript\nPOST /api/streaming/generate-link\n{\n  \"movieId\": \"12345\",\n  \"sourceUrl\": \"https://servidor-origen.com/video/12345\"\n}\n\n// Respuesta\n{\n  \"success\": true,\n  \"url\": \"https://stream-server.com/video/12345?token=abc123&t=1234567890\",\n  \"expires\": 1234567890,\n  \"movieId\": \"12345\"\n}\n```\n\n### Generar Múltiples Enlaces\n```javascript\nPOST /api/streaming/generate-multiple\n{\n  \"requests\": [\n    { \"movieId\": \"1\", \"sourceUrl\": \"https://server1.com/1\" },\n    { \"movieId\": \"2\", \"sourceUrl\": \"https://server2.com/2\" }\n  ]\n}\n\n// Respuesta\n{\n  \"results\": [\n    { \"movieId\": \"1\", \"url\": \"...\", \"success\": true },\n    { \"movieId\": \"2\", \"success\": false, \"error\": \"Token no encontrado\" }\n  ]\n}\n```\n\n## Configuración\n\n### Variables de Entorno\n```bash\n# Backend\nPORT=5000\nSUPABASE_URL=your_supabase_url\nSUPABASE_ANON_KEY=your_supabase_key\n\n# Frontend\nREACT_APP_API_URL=http://localhost:5000/api\n```\n\n### Patrones de Extracción\n\nEl sistema detecta automáticamente estos patrones en el HTML:\n- `file: \"url\"`\n- `source: \"url\"`\n- `url: \"url\"`\n- `src: \"url.m3u8\"`\n- `src: \"url.mp4\"`\n\n## Características\n\n### ✅ Implementado\n- Extracción automática de tokens\n- Cache inteligente (15 min)\n- Soporte múltiples formatos\n- Panel de administración\n- API REST completa\n- Manejo de errores\n\n### 🔄 Mejoras Futuras\n- Rotación de proxies\n- Detección de captcha\n- Múltiples patrones por servidor\n- Estadísticas avanzadas\n- Rate limiting inteligente\n\n## Instalación\n\n### Backend\n```bash\ncd backend\nnpm install\nnpm run dev\n```\n\n### Frontend\n```bash\ncd frontend\nnpm install\nnpm start\n```\n\n## Troubleshooting\n\n### Error: \"No se pudo extraer el token\"\n- Verificar que la URL del servidor sea correcta\n- Comprobar que el servidor esté disponible\n- Revisar los patrones de extracción\n\n### Error: \"Cache lleno\"\n- Limpiar cache: `POST /api/streaming/cache/clean`\n- Verificar estadísticas: `GET /api/streaming/cache/stats`\n\n### Error de CORS\n- Verificar configuración de CORS en el backend\n- Comprobar que las URLs coincidan\n\n## Seguridad\n\n- ✅ Validación de URLs\n- ✅ Timeout en requests (10s)\n- ✅ Rate limiting por IP\n- ✅ Sanitización de inputs\n- ⚠️ Usar HTTPS en producción\n- ⚠️ Implementar autenticación para admin\n\n## Monitoreo\n\n```javascript\n// Estadísticas del cache\nGET /api/streaming/cache/stats\n\n// Respuesta\n{\n  \"size\": 45,\n  \"entries\": [\"movie_1_url1\", \"movie_2_url2\", ...]\n}\n```\n\nEste sistema automatiza completamente el proceso de generación de enlaces, proporcionando una experiencia fluida tanto para usuarios como administradores.
